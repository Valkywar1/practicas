Descripción

El término "diseño web responsivo" (Responsive Web Design, RWD) fue acuñado por el diseñador y desarrollador Ethan Marcotte en su artículo de mayo de 2010 en A List Apart, titulado "Responsive Web Design". Marcotte propuso un enfoque basado en tres principios clave: Fluid Grid Layouts: Usar un sistema de cuadrícula flexible basado en porcentajes en lugar de unidades fijas. Flexible Images: Escalar imágenes y medios en función del tamaño de su contenedor. Media Queries: Utilizar CSS3 para aplicar diferentes estilos según las características del dispositivo, como el tamaño de la pantalla.

Tras la publicación del artículo de Marcotte, el diseño responsivo comenzó a ganar popularidad rápidamente. Los navegadores web mejoraron su soporte para CSS3 y media queries, lo que facilitó la implementación de diseños responsivos.

Este libro es adecuado para cualquiera que tenga interés en aprender a desarrollar sus propias páginas web. No son necesarios conocimientos previos para aprender con este libro, lo único que es necesario es saber utilizar un ordenador y saber navegar por la Web.

Contenido del libro

El contenido de este libro se estructura en tres apartados bien diferenciados:

  • viewport que es la área visible del contenido web en una pantalla. En diseño responsivo, el viewport se ajusta a diferentes tamaños de pantalla usando la etiqueta en el HTML
  • Media Queries. Las media queries son una característica de CSS3 que permite aplicar estilos diferentes en función de las características del dispositivo, como el tamaño de la pantalla, la resolución y la orientación. Se usan para adaptar el diseño a diferentes tamaños de pantalla.
  • Las cuadrículas flexibles utilizan porcentajes y unidades relativas en lugar de unidades fijas como píxeles. Esto permite que los elementos de la página se redimensionen proporcionalmente al tamaño del viewport. CSS Grid y Flexbox son herramientas comunes para crear layouts flexibles.
